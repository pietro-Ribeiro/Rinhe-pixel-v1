# Rinhe pixel (v1)
Um jogo de cartas de galos feito por Pietro Ribeiro, vença de galos bebados na rua e lute em rinhas de elite, consiga moedas e mais!

### Tecnologias usadas :
* HTML5 (sem tags semânticas mas com dialog)
* Css (com flexbox)
* javaScript (com localStorage, POO e Math)

## coisas do jogo :
* Tags dialog como notificação de vitória, derrota e empate
* Modo escuro e claro nas configurações do jogo
* Moedas permanentes com localStorage
* Modo briga de rua (fácil)
* Modo rinha de elite (difícil)

### Cronograma de versões:
* v1 : luta básica, moedas funcionando
* v2 : loja básica (+1 galo), css refinado
* v3 : ranking local, loja (+1 galo)
